### How to Run the Project Locally

### Requirements:
 - Node.js and npm installed
 - Git installed
 - MongoDB installed

### Job Board Website:
## 1. Clone the Repository in Git Bash-
  git clone https://github.com/your-username/CODSOFT.git
  cd CODSOFT
## 2. Switch to "master" Branch
  git checkout master
## 3. Start The Backend
  cd server
  npm install
  node index.js
## 4. Start Frontend
  cd ../client
  npm install
  npm start
